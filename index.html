氣泡排序法 (Bubble Sort)

說明

氣泡排序法是一種簡單的排序演算法。它重複地走訪列表，比較相鄰的元素，如果它們的順序錯誤就交換它們。這個過程一直持續到列表排序完成。

演算法特性：

  • 時間複雜度：O(n²)
  • 空間複雜度：O(1)
  • 類型：穩定排序

Python 程式碼範例

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr
