選擇排序法 (Selection Sort)

說明

選擇排序法是一種簡單的排序演算法。它將陣列分成已排序和未排序兩部分，每次從未排序部分選擇最小（或最大）的元素，放到已排序部分的末尾。

演算法特性：

  • 時間複雜度：O(n²)
  • 空間複雜度：O(1)
  • 類型：不穩定排序

Python 程式碼範例

def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        if min_idx != i:
            arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr
