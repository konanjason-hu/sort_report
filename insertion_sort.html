插入排序法 (Insertion Sort)

說明

插入排序法是一種簡單的排序演算法。它將陣列分成已排序和未排序兩部分，每次從未排序部分取一個元素，插入到已排序部分的正確位置。

演算法特性：

  • 時間複雜度：O(n²)
  • 空間複雜度：O(1)
  • 類型：穩定排序

Python 程式碼範例

def insertion_sort(arr):
    n = len(arr)
    for i in range(1, n):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
