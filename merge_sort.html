合併排序法 (Merge Sort)

說明

合併排序法是一種分治排序演算法。它先將陣列不斷拆分成子陣列，然後再將已排序的子陣列合併起來，最後得到完整排序陣列。

演算法特性：

  • 時間複雜度：O(n log n)
  • 空間複雜度：O(n)
  • 類型：穩定排序

Python 程式碼範例

def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
